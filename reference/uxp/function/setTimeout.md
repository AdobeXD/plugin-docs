## setTimeout(callback, delta)
Creates a request to call `callback` after `delta` milliseconds. The
request is cancelable using the return ID.

**Returns**: `\*` - Timer ID

**Category**: browser, timers

| Param | Type |
| --- | --- |
| callback | `function` |
| delta | `number` |
